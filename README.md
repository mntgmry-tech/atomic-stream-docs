# AtomicStream Docs

Documentation site for AtomicStream x402 streams.

Live site: https://docs.atomicstream.net

## Requirements

- Node.js 20+
- npm

## Local development

```bash
npm install
npm run dev
```

VitePress will serve the site at the URL shown in the terminal.

## Build

```bash
npm run build
npm run preview
```

Build output is generated in `docs/.vitepress/dist/`.

## Content

Documentation lives under `docs/`.

- Home: `docs/index.md`
- x402 docs: `docs/x402/`
- Stream catalog: `docs/streams/`
- Examples: `docs/examples/`
- Coming soon: `docs/coming-soon/`
